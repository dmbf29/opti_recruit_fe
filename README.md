## Getting Started
### Setup

Install gems
```
bundle install
```
Install JS packages
```
yarn install
```

### DB Setup
```
rails db:create
rails db:migrate
rails db:seed
rails heroku:pg_pull # requires admin access
```

### Run a server
```
rails s
```
